Для того чтобы система заработала нужно.

1. Запустить игру.
2. Добавить предмет в инвентарь. На объекте Inventory в Hierarchy в класс InventoryContex нужно положить конфиги из папке configs.
3. Ввести название добавленного предмета в классe ItemApplier и нажать Invoke. Этот класс применяет предмет игроку. Для того чтобы вернуть предмет в инвентарь нужно ввести данные имени в ReturnItemInventory и нажать Invoke.
